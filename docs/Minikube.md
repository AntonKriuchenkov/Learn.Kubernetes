# Using Minikube

Minikube is a single node kubernetes cluster, which can spinup k8s cluster on linux host using docker, virtualbox or any other hypervisor.

Current learn topick will be performaed on windows host, with installed Virtualbox and linux virtual machime (VM).
Primery user interface used for VM interactions is cli SSH.

Setup plan:

- Install Virtualbox
- Create VM
- Configure VM network and SSH
- Install minikube on VM
- Install kubectl on VM
- Test minikube and kubectl
